# Code of Conduct

## Our pledge

We want this project to be a welcoming, harassment-free space for everyone — regardless of experience level, background, identity, or affiliation.

## Expected behavior

- Be respectful and constructive in issues, PRs, and discussions.
- Assume good intent. If something looks wrong, ask before accusing.
- Keep feedback focused on the contribution, not the contributor.
- Respect maintainer decisions, even when you disagree.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Spam, self-promotion disguised as contributions, or SEO manipulation.
- Publishing others' private information.
- Trolling or deliberately derailing discussions.
- Any behavior that would make a reasonable person feel unwelcome.

## Enforcement

Maintainers will remove, edit, or reject contributions that violate this code of conduct. Repeat or egregious violations will result in being blocked from the repository.

This is a small, community-maintained project. We don't have a formal moderation team — maintainers make judgment calls as needed.

## Scope

This code of conduct applies to all project spaces: issues, pull requests, discussions, and any other communication channel associated with this repository.
