# Contributing to Awesome Finance APIs

Thanks for your interest in contributing! This list thrives on community input.

## How to propose a new API

1. **Check for duplicates.** Search the README and open issues/PRs to make sure the API isn't already listed or under review.
2. **Open a Pull Request** adding the entry to the appropriate category in `README.md`.
3. **Use the entry template** below so the list stays consistent.

## Entry template

```markdown
- [**API Name**](https://link-to-docs-or-homepage) — One-sentence description of what the API provides.
  - **Best for:** Primary use case (e.g., "retail equity backtesting").
  - **Free tier:** Brief summary of limits (e.g., "500 req/day, 5 yrs history").
  - **Limitations:** Notable constraints (e.g., "delayed quotes only", "non-commercial license").
```

### Field guidelines

| Field | Required | Notes |
|---|---|---|
| Name + link | Yes | Link to official docs or homepage; no affiliate links. |
| Description | Yes | One sentence. Avoid marketing copy. |
| Best for | Yes | Who benefits most from this API? |
| Free tier | Yes | Approximate limits are fine if the provider doesn't publish exact numbers. |
| Limitations | No | Only if there's something a developer should know upfront. |

## Other ways to contribute

- **Fix a broken link** — Open a PR or issue.
- **Improve a description** — Clarity and accuracy are always welcome.
- **Suggest a new category** — Open an issue to discuss before adding.
- **Report an API that has removed its free tier** — Open an issue so we can update or remove the entry.

## Guidelines

- Keep entries factual and concise.
- Do not add APIs that require paid plans with no usable free/freemium tier.
- Do not add APIs that are defunct, permanently down, or abandoned (no updates in 2+ years with known issues).
- Sort entries alphabetically within each category.
- One API per PR makes reviews faster.

## Code of Conduct

Be respectful and constructive. We follow common open-source community norms — no harassment, spam, or self-promotion disguised as contributions.
