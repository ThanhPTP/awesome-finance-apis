# Security Policy

## About this repository

This is a **documentation-only repository** — it contains a curated list of links and descriptions in Markdown files. There is no application code, no server, no database, and no user data.

## Reporting

There is no security reporting process for this repository because there is nothing to exploit. If you find a broken or suspicious link, please [open an issue](../../issues/new) instead.

## Scope

This repository does not handle authentication, secrets, PII, or any sensitive data. The APIs listed here are third-party services — any security concerns about those services should be reported directly to their respective maintainers.
